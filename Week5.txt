Pointer
--------------------------------------
ตัวแปร Pointer เป็นการ กำหนดค่าเพื่อให้ตัวแปรใช้สำหรับเก็บค่า ตำแหน่ง(Address)ของ Memory นั้นๆ
int *p; /*(* = Dereferencing Symbol)*/

p = 10;
หมายความว่า p ชี้ไปที่ Memory Address #10 ซึ่งใน Address 10 มีข้อมูลอยู่
จะนำข้อมูลข้างในออกมาแสดงให้ Dereference Pointer
